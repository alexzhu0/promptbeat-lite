# promptbeat-lite v0.2.1

Trend-aligned release for prompt eval and agent regression workflows.

## Added

- Directory suite mode for multiple JSON fixture files.
- Golden file support through `golden_file`.
- Basic fixture schema validation for missing output or expectation fields.
- JUnit plus fail-under examples for CI use.

## Try It

```bash
PYTHONPATH=src python3 -m promptbeat_lite examples/suite --format junit --fail-under 100
```
