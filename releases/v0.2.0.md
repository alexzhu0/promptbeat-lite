# promptbeat-lite v0.2.0

## What Changed

- Added pass-rate scoring and fail-under thresholds.
- Added JUnit output.
- Added compatibility with mined eval-case fixtures.

## Quickstart

```bash
PYTHONPATH=src python3 -m promptbeat_lite examples/cases.json --fail-under 100
```

## Verification

```bash
PYTHONPATH=src python3 -m unittest discover -s tests
```
